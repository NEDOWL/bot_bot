# MedBot — Telegram-бот для онлайн-консультаций с врачами
![Обложка](images/cover.png)

Telegram-бот, который объединяет врачей и пациентов на одной платформе. Пациенты могут найти специалиста, записаться на консультацию и общаться с врачом в чате. Врачи регистрируются, проходят верификацию документов и принимают заявки на приём.

## Возможности

### Для пациентов
- Регистрация профиля (имя, телефон, email)
- Поиск врачей по 14 направлениям: терапевт, педиатр, кардиолог, невролог, гинеколог и др.
- Сортировка врачей по рейтингу, цене и стажу
- Просмотр карточки врача с ценой и описанием
- Запись на консультацию с описанием проблемы
- Чат с врачом: текст, фото, видео и голосовые сообщения

### Для врачей
- Регистрация и редактирование профиля
- Верификация через загрузку фото документов (проверка администратором)
- Выбор одной или нескольких специальностей
- Установка стоимости консультации по каждому направлению
- Подтверждение или отклонение заявок от пациентов
- Ведение консультаций в чате с пациентом

### Администрирование
- Модерация заявок на верификацию врачей (принять / отклонить)
- Поддержка оплаты через CloudPayments (интеграция в разработке)

## Технологии

- **Python 3**
- **pyTelegramBotAPI** (`telebot`) — работа с Telegram Bot API
- **SQLite** — хранение пользователей, консультаций, чатов и транзакций
- **CloudPayments** — приём платежей за консультации

## Структура проекта

| Файл | Назначение |
|------|------------|
| `main.py` | Основная логика бота: регистрация, профили, чаты, консультации |
| `db.py` | Создание и инициализация базы данных SQLite |
| `test_doc.py` | Скрипт для добавления тестовых данных врачей |
| `del.py` | Утилита для очистки временных данных |

## Запуск

1. Установите зависимости:

```bash
pip install pyTelegramBotAPI requests
```

2. Создайте базу данных:

```bash
python db.py
```

3. Укажите токен бота и ID администратора в `main.py` (переменные `TOKEN` и `ADMIN_ID`).

4. Запустите бота:

```bash
python main.py
```

## Как пользоваться

1. Отправьте боту команду `/start`.
2. Выберите роль — **Доктор** или **Пациент**.
3. Заполните данные регистрации.
4. **Пациент:** нажмите «Записаться к врачу» → выберите специальность → выберите врача → опишите проблему.
5. **Врач:** пройдите верификацию, выберите специальности, укажите цены и подтверждайте заявки пациентов.
6. После подтверждения консультации откройте чат через раздел «Чаты».

## База данных

Бот использует SQLite (`db.db`) со следующими таблицами:

- `doctors` — профили врачей
- `patients` — профили пациентов
- `specialisation` — специальности и цены врачей
- `consultations` — заявки и записи на консультации
- `chats` — переписка между врачом и пациентом (сообщения хранятся в JSON)
- `transactions` — платёжные операции
- `reviews` — отзывы и оценки
